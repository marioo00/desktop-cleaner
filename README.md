### **How to Run**

1. Open **Terminal** (macOS or Linux).
2. Navigate to the folder where the script is located:

   ```bash
   cd /path/to/script
   ```
3. Run the script using Python 3:

   ```bash
   python3 file_cleaner.py
   ```

   Or make it executable:

   ```bash
   chmod +x file_cleaner.py
   ./file_cleaner.py
   ```
4. When prompted, enter the directory you want to clean, for example:

   ```
   /Users/yourname/Desktop
   ```
5. The script will:

   * List all files in that directory
   * Ask for confirmation
   * Automatically sort the files into folders like **Images**, **Documents**, **Music**, **Powerpoint**, or **Others**
